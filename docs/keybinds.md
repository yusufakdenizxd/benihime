## Default Keybindings

### Normal Mode

- `i` → Enter Insert mode
- `:` → Enter Command mode
- `v` → Enter Visual mode
- `h j k l` → Move left / down / up / right
- `w` → Next word
- `b` → Previous word
- `e` → Word end
- `0` → Beginning of line
- `-` → First non-blank character
- `=` → End of line
- `o` → Open line below
- `O` → Open line above
- `d` → Delete range
- `c` → Change range
- `zz` → Center cursor
- `Shift + j` → Next buffer
- `Shift + k` → Previous buffer
- `Cmd + p` → Find file
- `Cmd + b` → Find buffer
- `Cmd + x` → Open minibuffer
- `Ctrl + q` → Kill current buffer

---

### Insert / Command / Minibuffer Modes

- `Esc` → Return to Normal mode

---

### Command Mode

- `Enter` → Execute command

---

### Visual Mode

- `v` / `Esc` → Exit Visual mode
- `o` → Switch selection end
- `d` → Delete selection
- `c` → Change selection
- `h j k l` → Move selection
- `w b e` → Word motions

---

### Minibuffer Mode

- `Enter` → Accept
- `Ctrl + j` / `Down` → Next completion
- `Ctrl + k` / `Up` → Previous completion
